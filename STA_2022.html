📌 Q1 Convergence of Real Sequences (10 Marks)

Problem Statement: Consider the sequence of real numbers:
$$x_n = \frac{(\alpha + 1)^n + \alpha^n + (\alpha - 1)^n}{(2\alpha)^n}, \quad n = 1, 2, \dots$$
Find the real values of $\alpha$ for which $\lim_{n \to \infty} x_n$ exists.

🧠 Approach & Key Concepts

This problem evaluates the convergence of sequences formed by sums of exponential terms. By algebraically splitting the fraction, we convert the expression into a sum of three geometric sequences: $r_1^n + r_2^n + r_3^n$. A sequence of the form $r^n$ converges if and only if $r \in (-1, 1]$. We will extract the three common ratios, set up simultaneous inequalities for their convergence, and find the intersection of their valid domains.

✍️ Step-by-Step Proof / Derivation

Step 1: Algebraic Decomposition

First, note that for the sequence to be defined, the denominator cannot be zero, hence $\alpha \neq 0$. We split the fraction into three separate terms:

$$x_n = \left(\frac{\alpha + 1}{2\alpha}\right)^n + \left(\frac{\alpha}{2\alpha}\right)^n + \left(\frac{\alpha - 1}{2\alpha}\right)^n$$
$$x_n = \left(\frac{\alpha + 1}{2\alpha}\right)^n + \left(\frac{1}{2}\right)^n + \left(\frac{\alpha - 1}{2\alpha}\right)^n$$

Let $r_1 = \frac{\alpha+1}{2\alpha}$, $r_2 = \frac{1}{2}$, and $r_3 = \frac{\alpha-1}{2\alpha}$. The term $r_2^n = (1/2)^n$ strictly converges to 0. For the entire sequence to converge without oscillating or exploding to infinity, we must strictly have both $r_1 \in (-1, 1]$ and $r_3 \in (-1, 1]$.

Step 2: Bounding $r_1$

We need $-1 < \frac{\alpha + 1}{2\alpha} \leq 1$. Let's solve the two inequalities separately.

Inequality 1A (Upper Bound):

$$\frac{\alpha + 1}{2\alpha} \leq 1 \implies \frac{\alpha + 1}{2\alpha} - 1 \leq 0 \implies \frac{1 - \alpha}{2\alpha} \leq 0$$

The critical points are $\alpha = 0$ and $\alpha = 1$. Testing intervals yields the solution: $\alpha \in (-\infty, 0) \cup [1, \infty)$.

Inequality 1B (Lower Bound):

$$-1 < \frac{\alpha + 1}{2\alpha} \implies 0 < \frac{\alpha + 1}{2\alpha} + 1 \implies 0 < \frac{3\alpha + 1}{2\alpha}$$

The critical points are $\alpha = -1/3$ and $\alpha = 0$. Testing intervals yields the solution: $\alpha \in (-\infty, -1/3) \cup (0, \infty)$.

Taking the intersection of 1A and 1B, the valid region for $r_1$ is: $S_1 = (-\infty, -1/3) \cup [1, \infty)$.

Step 3: Bounding $r_3$

We need $-1 < \frac{\alpha - 1}{2\alpha} \leq 1$.

Inequality 3A (Upper Bound):

$$\frac{\alpha - 1}{2\alpha} \leq 1 \implies \frac{\alpha - 1}{2\alpha} - 1 \leq 0 \implies \frac{-1 - \alpha}{2\alpha} \leq 0 \implies \frac{\alpha + 1}{2\alpha} \geq 0$$

Critical points are $\alpha = -1, 0$. The solution is: $\alpha \in (-\infty, -1] \cup (0, \infty)$.

Inequality 3B (Lower Bound):

$$-1 < \frac{\alpha - 1}{2\alpha} \implies 0 < \frac{\alpha - 1}{2\alpha} + 1 \implies 0 < \frac{3\alpha - 1}{2\alpha}$$

Critical points are $\alpha = 0, 1/3$. The solution is: $\alpha \in (-\infty, 0) \cup (1/3, \infty)$.

Taking the intersection of 3A and 3B, the valid region for $r_3$ is: $S_3 = (-\infty, -1] \cup (1/3, \infty)$.

Step 4: Final Intersection

The sequence converges if and only if $\alpha$ is in both $S_1$ and $S_3$.

$$S_{final} = \Big( (-\infty, -1/3) \cup [1, \infty) \Big) \cap \Big( (-\infty, -1] \cup (1/3, \infty) \Big)$$
  • For the negative intervals: $(-\infty, -1/3) \cap (-\infty, -1] = (-\infty, -1]$.
  • For the positive intervals: $[1, \infty) \cap (1/3, \infty) = [1, \infty)$.
Final Answer / Q.E.D: The limit exists for all real numbers $\alpha \in (-\infty, -1] \cup [1, \infty)$, which can also be written compactly as $|\alpha| \geq 1$.

📌 Q2 Range and Extrema of a Trigonometric Function (10 Marks)

Problem Statement: For $x \geq 0$, define $f(x) = x - \sqrt{2}\sin(x)$, with $x$ in radians. Find the range of the function $f$.

🧠 Approach & Key Concepts

To find the range of a continuous function on a semi-infinite interval $[0, \infty)$, we must determine its global minimum and examine its behavior as $x \to \infty$. We find the critical points by setting the derivative $f'(x) = 0$. Because the function comprises a linearly growing term ($x$) and a bounded oscillating term ($\sin(x)$), it will naturally tend toward infinity, meaning there is no upper bound. The challenge lies in proving which local minimum is the absolute global minimum.

✍️ Step-by-Step Proof / Derivation

Step 1: Finding Critical Points

We compute the first derivative of $f(x)$:

$$f'(x) = 1 - \sqrt{2}\cos(x)$$

Set $f'(x) = 0$ to find the critical points:

$$1 - \sqrt{2}\cos(x) = 0 \implies \cos(x) = \frac{1}{\sqrt{2}}$$

Since we are restricted to $x \geq 0$, the general solutions are:

$$x = \frac{\pi}{4} + 2k\pi \quad \text{and} \quad x = \frac{7\pi}{4} + 2k\pi \quad \text{for } k = 0, 1, 2, \dots$$

Step 2: Identifying Local Minima

We use the second derivative test to classify these critical points:

$$f''(x) = \sqrt{2}\sin(x)$$
  • At $x_k = \frac{\pi}{4} + 2k\pi$, $f''(x_k) = \sqrt{2}\sin(\pi/4) = \sqrt{2}(1/\sqrt{2}) = 1 > 0$. These are local minima.
  • At $x_k = \frac{7\pi}{4} + 2k\pi$, $f''(x_k) = \sqrt{2}\sin(7\pi/4) = -1 < 0$. These are local maxima.

Step 3: Determining the Global Minimum

We evaluate the function at the sequence of local minima:

$$f\left(\frac{\pi}{4} + 2k\pi\right) = \left(\frac{\pi}{4} + 2k\pi\right) - \sqrt{2}\sin\left(\frac{\pi}{4}\right) = \frac{\pi}{4} + 2k\pi - 1$$

Notice that as $k$ increases, the value of the local minimum strictly increases by $2\pi$. Therefore, the lowest possible value among all local minima occurs at $k = 0$:

$$f\left(\frac{\pi}{4}\right) = \frac{\pi}{4} - 1$$

We must also check the boundary point at $x = 0$:

$$f(0) = 0 - \sqrt{2}\sin(0) = 0$$

Comparing the two, since $\pi \approx 3.14$, we have $\frac{\pi}{4} - 1 \approx 0.785 - 1 = -0.215 < 0$. Thus, $f(\pi/4)$ is strictly less than $f(0)$, making it the absolute global minimum.

Step 4: Upper Bound Behavior

As $x \to \infty$, the linear term $x$ grows without bound, while the $\sqrt{2}\sin(x)$ term remains strictly bounded between $[-\sqrt{2}, \sqrt{2}]$. Therefore:

$$\lim_{x \to \infty} f(x) = \infty$$

Since $f$ is continuous on $[0, \infty)$, it must attain all values between its global minimum and infinity (by the Intermediate Value Theorem).

Final Answer / Q.E.D: The global minimum is $\frac{\pi}{4} - 1$ and the function grows without bound. Therefore, the range of the function $f$ is exactly $\left[ \frac{\pi}{4} - 1, \infty \right)$.

📌 Q3 Expectation of Symmetric Ratios (10 Marks)

Problem Statement: Let $X_1, X_2$ and $X_3$ be iid random variables with a pdf symmetric about 0. Assuming that moments of all orders exist, find the value of: $$E\left( \frac{X_1^3 - X_2^3 + X_3^3}{\sqrt{X_1^2 + X_2^2 + X_3^2}} \right)$$

🧠 Approach & Key Concepts

This problem evaluates your intuition regarding Symmetry in Probability Distributions. Trying to compute this expectation using integrals and joint density functions would be practically impossible. Instead, we define the entire expression as a single transformation $Y = g(X_1, X_2, X_3)$ and exploit the mathematical property that if a random variable $Y$ is symmetrically distributed about 0 (i.e., $Y \stackrel{d}{=} -Y$) and its expectation exists, its expectation must exactly equal zero.

✍️ Step-by-Step Proof / Derivation

Step 1: Establishing the Symmetric Joint Distribution

We are given that $X_1, X_2, X_3$ are independent and identically distributed (i.i.d.) with a pdf symmetric about 0. This symmetry means that for any individual variable, $X_i \stackrel{d}{=} -X_i$ (they have the exact same distribution).

Because they are independent, their joint distribution is also invariant under negation. That is, the random vector $(X_1, X_2, X_3)$ has the exact same joint distribution as $(-X_1, -X_2, -X_3)$:

$$(X_1, X_2, X_3) \stackrel{d}{=} (-X_1, -X_2, -X_3)$$

Step 2: Defining the Transformation

Let the target random variable be $Y$:

$$Y = g(X_1, X_2, X_3) = \frac{X_1^3 - X_2^3 + X_3^3}{\sqrt{X_1^2 + X_2^2 + X_3^2}}$$

We now evaluate how this function behaves when we plug in the negated vector $(-X_1, -X_2, -X_3)$:

$$g(-X_1, -X_2, -X_3) = \frac{(-X_1)^3 - (-X_2)^3 + (-X_3)^3}{\sqrt{(-X_1)^2 + (-X_2)^2 + (-X_3)^2}}$$

Because the exponent in the numerator is odd (3), the negative sign factors out perfectly. Because the exponent in the denominator is even (2), the negative sign vanishes:

$$= \frac{-X_1^3 - (-X_2^3) - X_3^3}{\sqrt{X_1^2 + X_2^2 + X_3^2}} = \frac{-(X_1^3 - X_2^3 + X_3^3)}{\sqrt{X_1^2 + X_2^2 + X_3^2}} = -g(X_1, X_2, X_3) = -Y$$

Step 3: Drawing the Statistical Conclusion

Because the input vectors are identically distributed, any deterministic function applied to them will yield identically distributed outputs. Thus:

$$g(X_1, X_2, X_3) \stackrel{d}{=} g(-X_1, -X_2, -X_3)$$

Substituting our result from Step 2, we get:

$$Y \stackrel{d}{=} -Y$$

This strictly proves that the random variable $Y$ itself is symmetrically distributed around 0. The problem guarantees that all moments exist, guaranteeing that $\mathbb{E}(Y)$ is well-defined and finite.

For any random variable where $Y \stackrel{d}{=} -Y$, we have $\mathbb{E}(Y) = \mathbb{E}(-Y) = -\mathbb{E}(Y)$. The only finite number that is its own negative is 0.

Final Answer / Q.E.D: By proving that the transformation function is strictly odd, we established that $Y$ is symmetrically distributed about zero. Therefore, $E(Y) = 0$.
